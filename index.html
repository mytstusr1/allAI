🤖 AI & ML for Kids

A Complete 3-Year Teaching Curriculum for Ages 13–15

Age 13: Discovery & Exploration

Foundation Year

12 Weeks Duration
6 Modules Topics
No Coding Required

✨ What's Different at This Age

Age 13 is about discovery and wonder. No coding knowledge required. We focus on understanding AI through interactive tools, hands-on exploration, and real-world examples. Students build confidence and develop a strong ethical foundation.

🧠
What is AI?
Weeks 1–2 (2 weeks)

Topics

  • AI vs Human Intelligence
  • Types of AI (Narrow vs General)
  • Real-world examples

Activities

  • Discussion: "Which tasks can AI do better?"
  • Sort real-world examples (Siri, chess bots, self-driving cars)
  • Reflect on AI in daily life
Tools: Day of AI (dayofai.org)
📖
How Machines Learn
Weeks 3–4 (2 weeks)

Topics

  • Supervised Learning
  • Unsupervised Learning
  • Training Data & Overfitting

Activities

  • Card sorting game
  • Hands-on with Teachable Machine
  • Train model & test accuracy
Tools: teachablemachine.withgoogle.com
👁️
Seeing, Hearing & Understanding
Weeks 5–6 (2 weeks)

Topics

  • Computer Vision
  • NLP & Chatbots
  • ML for Kids platform

Activities

  • Image classifier with ML for Kids
  • Build simple chatbot in Scratch
  • Mini-project: Choose one
Tools: ML for Kids, Scratch ML extension
⚖️
AI Ethics
Weeks 7–8 (2 weeks)

Topics

  • Bias in AI
  • Privacy & Security
  • Jobs & Power

Activities

  • Facial recognition bias case study
  • Class debate: "AI helpful vs harmful"
  • Moral Machine interactive tool
Tools: Moral Machine, Gender Shades project
🛠️
Build Your Own AI Project
Weeks 9–10 (2 weeks)

Options

  • Image Classifier
  • Smart Chatbot
  • Sound Recognizer
  • Social Good proposal

Process

  • Idea & Planning
  • Data Collection
  • Training & Testing
  • Improving & Iteration
Tools: Teachable Machine, ML for Kids, Scratch
🎤
Present & Reflect
Weeks 11–12 (2 weeks)

Activities

  • Science Fair demo day
  • Live Q&A with peers
  • Reflection & discussion

Assessment

  • Live demo + Q&A
  • Video explanation
  • Written reflection
Tools: All tools used throughout the year

Python Programming

Age 14 bridges the gap between exploration and creation. Students learn Python programming in the context of machine learning. We focus on building real projects, understanding algorithms, and deepening ethical thinking. This is where students become creators, not just consumers, of AI.

⚙️ Module 2: ML Pipeline with GridSearch
from sklearn.ensemble import RandomForestClassifier from sklearn.model_selection import GridSearchCV from sklearn.metrics import classification_report X_train, X_test, y_train, y_test = train_test_split( X, y, test_size=0.2) param_grid = {"n_estimators": [50, 100, 200], "max_depth": [3, 5, None]} grid = GridSearchCV(RandomForestClassifier(), param_grid, cv=5, scoring="f1") grid.fit(X_train, y_train) print("Best:", grid.best_params_) print(classification_report(y_test, grid.predict(X_test)))
🧠 Module 3: Keras Neural Network (MNIST)
import tensorflow as tf from tensorflow import keras (X_train, y_train), (X_test, y_test) = keras.datasets.mnist.load_data() X_train, X_test = X_train / 255.0, X_test / 255.0 model = keras.Sequential([ keras.layers.Flatten(input_shape=(28, 28)), keras.layers.Dense(128, activation="relu"), keras.layers.Dropout(0.2), keras.layers.Dense(64, activation="relu"), keras.layers.Dense(10, activation="softmax") ]) model.compile(optimizer="adam", loss="sparse_categorical_crossentropy", metrics=["accuracy"]) model.fit(X_train, y_train, epochs=10, validation_split=0.1) print(f"Test: {model.evaluate(X_test, y_test)[1]:.2%}")
👁️ Module 4: Transfer Learning with MobileNetV2
from tensorflow.keras.applications import MobileNetV2 from tensorflow.keras import layers, models base_model = MobileNetV2(weights="imagenet", include_top=False, input_shape=(224, 224, 3)) base_model.trainable = False model = models.Sequential([ base_model, layers.GlobalAveragePooling2D(), layers.Dense(3, activation="softmax") ]) model.compile(optimizer="adam", loss="sparse_categorical_crossentropy", metrics=["accuracy"])
✨ Module 4: NLP Sentiment with Transformers
from transformers import pipeline classifier = pipeline("sentiment-analysis") reviews = [ "This movie was absolutely amazing!", "Terrible film, total waste of time.", "It was okay, nothing special." ] for review in reviews: result = classifier(review)[0] print(f"{result['label']} ({result['score']:.0%})") print(f" → {review}")
